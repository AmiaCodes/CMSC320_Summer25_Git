Q1: Why might we consider using Git?

Answer: Git is the main version control system. Git tracks the changes you make to files for records and ability to revert changes. Git also makes collaboration easier, streamlining changes by multiple people into one source.

Q2: What field could data science be applied to? Give an Example.

Answer: Healthcare - data science can be used to predict disease outbreaks or tailor treatment plans for each individual.

Q3: What is the difference between Merging and Rebasing?

Answer: Merging is a safe option that preserves the entire history of your repository, while rebasing creates a linear history by moving your feature branch onto the tip of main.
